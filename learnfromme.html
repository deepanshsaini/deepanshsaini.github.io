Course Details


This is a beginner course with no prior coding experience required and open to all students who is looking forward to learn hacking (Ethical of-course!) or making a career out of it.

I will only teach 15 students at a time in a batch which will take almost 3 months to complete the course with weekend classes. Please fill the registration form above.

15 Students will be randomly selected from the students and the students who did not get selected will be reached out if I start any other batch or in future batches.

Course Curriculum 

The curriculum is as follow:-

Module 01 : Introduction to Cyber Security, Ethical Hacking, Networking and OS
Module 02 : Foot-printing Active and Passive
Module 03 : In Depth Network Scanning
Module 04 : Enumeration User Identification
Module 05 : System Hacking Password Cracking & Bypassing
Module 06 : Malware Threats
Module 07 : Sniffing
Module 08 : Vulnerability Assessments
Module 09 : Social Engineering Theoretical and Practical Approach
Module 10 : Denial of Service Attack
Module 11 : Web Session Hijacking
Module 12 : SQL Injection Manual and Automation Testing
Module 13 : Basics of Web App Security
Module 14 : Hacking Wireless Network CLI and UI Based
